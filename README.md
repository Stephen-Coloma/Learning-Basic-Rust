# Rust Basics Learning

This repository covers learning the basics of Rust, including:

- Data Types
- Functions
- Conditionals
- Control Flow
- Shadowing, Ownership and Borrowing
- Structs
- Traits
- Simple OOP Concepts

To run the examples, go to src directory and use:

```bash
cargo run
```
